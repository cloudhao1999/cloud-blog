Nginx配置域名转发

假如我输入一个www开头的域名，但是我的证书只认证了非www开头的域名，可以作一下配置

server {
	include blocksip.conf;
    if ($host = www.你的域名) {
        return 301 https://你的域名$request_uri;
    } 


	listen 80;
	listen [::]:80;

	server_name www.你的域名;
    return 404; 

}
